1. 개인정보 수집

물방울은 어떠한 개인정보도 수집하지 않습니다.

앱 사용에 필요한 모든 데이터(물 섭취 기록, 알림 설정 등)는 사용자의 기기에만 저장되며, 외부 서버로 전송되지 않습니다.

2. 데이터 저장

  • 물 섭취 기록: 기기 로컬 데이터베이스에 저장
  • 사용자 설정: 기기 내부 저장소에 저장
  • 알림 설정: 기기 시스템에 저장

3. 데이터 공유

물방울은 제3자와 어떠한 데이터도 공유하지 않습니다.

4. 권한 사용

  • 알림 권한: 물 마시기 알림을 보내기 위해서만 사용됩니다.

5. 데이터 삭제

앱을 삭제하면 모든 데이터가 자동으로 삭제됩니다.

6. 아동 개인정보보호
